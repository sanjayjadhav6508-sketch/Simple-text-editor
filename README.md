# Simple-text-editor


A simple and user-friendly text editor built using Python and Tkinter.

##  Features
- Create a new file
- Open existing text files
- Edit text easily
- Save files
- Clean and simple interface
- Beginner-friendly project

##  Technologies Used
- Python
- Tkinter
